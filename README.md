# Darts Premier League Manager Python Program

The PDC Premier League Darts format served as the inspiration for this Python-based tournament manager and simulator.

With this project, users can use the command line to manage players, enter match results, monitor statistics, and create a fully automated leaderboard with sophisticated metrics.

This system, which was initially developed as a learning exercise to hone Python skills, has developed into a fully functional competition manager with features that can be added for upcoming competitions.

-------------------------------------------------------------------------------

Features

Establish and oversee an eight-player Premier League structure

Enter the results of the match (automatic winner calculation)

Support for walkover (N/A score handling)

Comprehensive match results, night by night

-------------------------------------------------------------------------------

Automatic distribution of points:

Night Winner: 5 points

Runner-up: 3 points

Semi-finalists: 2 points

-------------------------------------------------------------------------------

Sorting dynamic leaderboards by:

Points

Difference in Legs

Comprehensive match results, night by night

Player search results

Players' head-to-head comparison

The majority of 180s tables

Calculating the win percentage

Using JSON for persistent data storage

Statistics Monitored

-------------------------------------------------------------------------------

Every player keeps track of:

Points

Games Played

Wins in Matches

Wins at Night

Winning Legs

Loss of Legs

Difference in Legs

Percentage of Wins

180s (expandable)
